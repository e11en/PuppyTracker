# PuppyTracker

A Home Assistant custom integration to track a puppy's first months: developmental
phases, a daily checklist, a socialization program, and shiftable care schedules
(defer a step and the rest of that schedule cascades along).

Built for Beer, a Bernese Mountain Dog. Runs as a custom integration (Python + SQLite)
with its own sidebar panel (Lit) — no add-on required.
